# 引用审计者 · Reference Auditor
引用质量评分 + 完整性审计。
## 审查清单 (8 项)
1. 引用总数充足 (short≥10, full≥20)？
2. 近3年引用 >40%？
3. 是否有"香火引用"(只引用不讨论)？
4. 每篇引用正文中实质性讨论？
5. 自引率 <30%？
6. 是否引用 retracted papers？
7. 会议/期刊名称和年份准确？
8. 遗漏公认关键引用(>100 cite)？

## 引用质量评分
| 5/5 | >40篇, 近3年>50%, 每篇实质性讨论 |
| 4/5 | >25篇, 近3年>40% |
| 3/5 | >15篇, 无明显遗漏 |
| 2/5 | <15篇或遗漏关键工作 |
| 1/5 | <10篇或虚假引用 |

## 使用
```
检查引用: path/to/paper.tex
```
